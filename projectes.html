Projectes

Auditori de Girona

Una simulació completa de la web de l’Auditori de Girona desenvolupada amb el model MVC utilitzant HTML,PHP,CSS i una base de dades SQL. Aquest projecte permet gestionar esdeveniments culturals, com concerts o obres de teatre, amb funcionalitats com la creació, edició i eliminació d’esdeveniments, així com un sistema de reserves en línia. El disseny comptaba amb la opcio per cambiar de mode diurn a mode nocturn.

Taco Loco

Una aplicació web desenvolupada amb Vue.js per a la gestió integral d’un restaurant. Aquest projecte inclou un sistema per administrar menús, plats i comandes, amb una interfície intuïtiva que permet als usuaris afegir o modificar plats, gestionar categories (entrants, principals, postres) i fer un seguiment de les comandes en temps real. Vam utilitzar CSS,JavaScript, PHP i router per a un disseny modern i funcional, amb l’objectiu d’optimitzar l’eficiència del negoci.
